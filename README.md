# Test-task

## Tech Stack
* React
* Redux
* TypeScript
* Styled-components

* ## How to run the app

1. Clone this repository by running `$ git@github.com:iliasen/Test-task.git`
2. Run `cd client`
3. Run `npm i`
4. Run `npm run dev`
